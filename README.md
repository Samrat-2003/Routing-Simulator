# Network Routing Simulation Project

## Project Overview
This project implements a comprehensive network routing simulation framework to analyze and compare different routing algorithms under various network conditions and traffic loads.

## Features
- Multiple network topologies (mesh, random, ring, star)
- Four routing algorithms:
  - Dijkstra's Algorithm
  - Bellman-Ford Algorithm
  - Ant Colony Optimization (ACO)
  - Genetic Algorithm (GA)
- Traffic generation with configurable intensity
- Performance metrics analysis
- Interactive web dashboard for visualization
- Comparative analysis of algorithms
